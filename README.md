#### Магазин будет состоять из следующих страниц (для продавцов):

  * Поля для ввода параметров 
* Список продукции (КАТАЛОГ МАГАЗИНА)

  * Редактировать продукт (По паролю)

    * Редактировать цену
    * Удалить продукт
* Личный кабинет 
  * Редактировать личную информацию магазина
